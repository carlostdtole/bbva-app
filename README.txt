App web desarrollada en JavaScript, HTML y CSS 
para la consulta del tiempo atmosférico de una determinada ciduad.
Además, se visualiza el pronóstico resumido de los próximos tres días.

Detalles prónostico: localización, icono tiempo actual, temperatura, 
estado del cielo, sensación atmosférica, humedad y velocidad del viento.

Detalles prónostico resumido: día, temperatura y icono tiempo.

Pasos realizados:
Creación del proyecto y archivos: index.html, script.js y style.css.
No se ha utilizado framework.

Instrucciones de uso:
1. Lanzar desde cualquier IDE el archivo index.html.
2. Introducir una ciudad o pulsar el botón de geolocalización.
3. Visualización de los datos.

Tests:
No se han realiado tests al incluir en el archivo script.js funciones de comprobación de errores.
